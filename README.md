# Primes Fun (iOS)
- Check whether a number is prime
- List the prime numbers in a range
- Factorize a number
- Generate random primes
- Share or save your results
- And more
- Get it here: https://itunes.apple.com/us/developer/daniel-springer/id1402417666?mt=8
